### v1.5（已 ship — 2026-06-26）✅ 實際完成
**注意**：實際 v1.5 與原 plan 規劃不同。原本 v1.5 規劃的 turtle 龜圖已**下移至 v1.6**；
v1.5 改為「無人機足球 + 大亂鬥多人 + 操控強化」功能包（提前開發，非綁定特定課程週次）。

**實際內容**：
- ⚽ 無人機足球單人練習（7 關 P-1~P-7：穿門 / 繞假人 / 來回穿梭 / 限時多穿；飛機含球形外框）
- 🏟️ 大亂鬥 Arena 多人即時連線（搶氣球 balloon / 鬼抓人 tag，**伺服器權威**，老師後台開賽、3-2-1 倒數、時間到/勝負結束）
- 🎮 藍牙搖桿（pyController Web Bluetooth）整合進主模擬器 + `/gamepad/` 連線測試頁
- 移動視覺傾斜 + 地面陰影（近正上方投影）+ 升版至 1.5.0

**Backlog（尚未完成）— ⚽ 足球「多人連線對戰」版**：
- **形式**：二隊各 3 人，共 6 人對戰（紅隊 vs 藍隊）
- **角色**：每隊分「攻擊手 / 防守手」，需在 UI 標示誰紅誰藍、誰攻誰守
- **得分**：操作自己的無人機（含球形外框）穿過**對方防守的球門**＝得分。
  **無共用球 — 無人機本身就是「球」**（對應真實無人機足球規則）
- **實作方向**：照已驗證的 Arena 伺服器權威模式新增 `soccer_start / soccer_go / soccer_pos / soccer_goal / soccer_end`，
  並加上隊伍/角色指派與顯示、穿門計分上報、開賽與結束（時間到比分判勝）流程
- 現況：足球只有**單人練習**（`SOCCER`，純前端、無連線），多人為 0

### v1.6（W3 — 7/6 前完成）
**目標**：第三堂課教 if / 迴圈 / 變數邏輯，6 個 turtle 邏輯關卡（drone 飛出龜圖）
**老師開場 5 分鐘**：講 logo 語言歷史（龜圖抽象化）+ 為什麼用「畫圖」學 loop
**核心載具**：drone Blockly 複刻 Python turtle 經典邏輯（畫正方形、星星、螺旋、圓形、六邊形）
**視覺特色**：drone 帶**噴霧**工具，移動時留下光帶（類似龜圖筆跡）— 學生看到圖形被「畫」出來
**不做**：距離感測類積木（詳見 ADR-001）

預計 task 群（T-201 ~ T-2xx，待 v1.4 完後細拆）：
- T-201: if / if-else 邏輯積木
- T-202: 迴圈積木（repeat N / while / forever）
- T-203: 變數積木（set / change）
- T-204: 時間型計時積木（`elapsed > 2s` 取代 `distance < 2m`）
- T-205: 6 個 turtle 邏輯關卡（3-0 正方形 / 3-1 星星 / 3-2 螺旋 / 3-3 圓形 / 3-4 六邊形 / 3-5 自選）
- T-206: **噴霧 paint 系統**（Blockly `paint on/off` + 3D 場景光帶渲染）
  - 學生拉 `paint on` 後，drone 移動時留下彩色光帶
  - 4 色可選（紅/藍/綠/黃）
  - 渲染：Three.js Line + glow shader，沿 drone 移動軌跡即時畫線
